加工精度が突然悪化したとき、原因を素早く特定できるかどうかが、生産ラインの稼働率を左右します。本記事では精度不良を「ワーク起因・チャック起因・機械起因」の3系統に分類し、測定器なしでも80%の原因を特定できるYes/No形式の診断フローチャートを提供します。各根本原因に対応した具体的な対策と、株式会社Sun Aiの製品による根本解決策もあわせて解説します。

## 026 加工精度±0.01mmが守れない！5分で原因を特定する診断フローチャート

### 1. なぜ「勘」より「フローチャート」が有効なのか

±0.01mmの公差外れは、数十種類の原因から発生しえます。経験頼りの診断では、誤った変数を調整し続けて機械を長時間停止させてしまうケースが後を絶ちません。構造化されたフローチャートは、熟練者の暗黙知を再現可能な手順に変換し、誰でも同じ品質で診断を実行できるようにします。

本記事が使う「3系統フレームワーク」は、あらゆる加工セットアップの物理的な現実を反映しています。

- **ワーク系**: 材料応力、表面状態、クランプによる変形
- **チャック系**: 保持力、残留磁気、平面度、位置決め繰り返し精度
- **機械系**: 幾何精度、熱変位、工具状態

どの系統が原因かを最初に特定することで、「別の誤差で補正する」という誤った対処を防ぎます。

---

### 2. 診断前に確認するベースラインデータ（所要時間：約2分）

フローチャートに入る前に、以下の項目を記録します。不明な項目があれば、その空白自体が診断の手がかりになります。

|確認項目|何を見るか|合否の目安|
|---|---|---|
|直前の良品|日付とロット番号|—|
|チャックON保持力|テストピース50×50mm換算|製品仕様値以上|
|チャックOFF残留磁気|チャック中央部をガウス計で計測|〜3〜5ガウス（SMシリーズ標準）|
|テーブル・チャック面平面度|ダイヤルゲージでスイープ|300mmあたり0.02mm以内|
|室温変動|温度ログ|加工中±1℃以内|

> **仕様確認メモ**: SWL-H190AB+はON時0.16T、OFF時0.008T。SWL-H190ABおよびSWL-H100αβはON時0.06T、OFF時0.006Tです。実測値が大きく乖離する場合はチャック系のブランチに進んでください。

---

### 3. 5分間診断フローチャート

各ブランチを順番に進みます。最初に「YES」と答えられた設問が、根本原因の系統を指し示します。

```
スタート：公差外れは毎回再現するか、それともランダムか？
│
├─ ランダム → 機械系ブランチへ（熱変位・振動・工具摩耗）
│
└─ 再現する → ずれの方向は一定か（常に+方向、または常に−方向）？
       │
       ├─ YES、方向一定 → チャック系ブランチへ（オフセット・平面度・残留磁気）
       │
       └─ NO、部位によってばらつく → ワーク系ブランチへ（応力・変形・表面状態）
```

#### ブランチA ― ワーク系

**Q1. チャックをOFFにしたとき、ワークが反り戻るか？**

- YES → 材料に内部応力がある。荒加工後に応力を解放してから仕上げ加工を実施すること。SM-Hシリーズのマグネットチャックは、岩手大学トライボロジー研究室との共同試験で、フライス加工中の「反ろう」力を抑制しながら加工できることが確認されており、平面度の高い仕上がりを実現します。
- NO → Q2へ。

**Q2. クランプ前にワーク表面のバリ・汚れは除去されているか？**

- NO → 表面を清掃する。わずか0.01mmのバリでもワークが傾き、目標公差と同等以上の誤差が生じます。
- YES → Q3へ。

**Q3. 薄物ワーク（板厚5mm以下）の場合、電極回転装置を使った追加工で真円度は改善するか？**

- YES → ワイヤー放電加工のみでは真円度が約0.0044mmの誤差を持ちます。SER-01A電極回転装置による追加工により、真円度を約0.0015mm、表面粗さをRa1.1646からRa0.1279に改善できることが加工比較試験（測定機器：Zeiss upmc-850、Accretech Surfcom1900sd3）で実証されています。
- NO → ワーク系が原因ではない。チャック系ブランチへ。

#### ブランチB ― チャック系

**Q1. チャックOFFの状態でダイヤルゲージをチャック面上でスイープし、300mmあたりの平面度が0.02mm以内か？**

- NO → チャック面が汚染または損傷している。徹底的に清掃する。清掃後も平面度が回復しない場合は株式会社Sun Aiに検査を依頼する。
- YES → Q2へ。

**Q2. チャックをOFFにした後、ガウス計でワーク接触面中央部が5ガウスを超えるか？**

- YES → 残留磁気が鉄粉を引き寄せ、次のワークの着座に影響している。SWL-H190ABシリーズはOFF時に脱磁機能が働くため、加工後の別途脱磁処理が不要です（特許第5716232号）。現在のチャックにこの機能がない場合はSWL-H190ABシリーズへの切り替えを検討してください。
- NO → Q3へ。

**Q3. チャックの保持力（テストポイント100mmでの荷重測定）は仕様値を満たしているか？**

- NO → 磁力が低下している。ON/OFFレバーの位置を確認し、磁極ピッチがワーク材質に対応しているか確認する（SMシリーズ標準は鉄：真鍮比 2:1）。それでも改善しない場合はチャックの交換が必要。
- YES → Q4へ。

**Q4. チャックはERWAまたはsystem3Rのパレットシステムに対応しており、パレットの繰り返し精度は公差内か？**

- NO → サイクルごとのパレット着座のばらつきが位置オフセットを生んでいる。SM-Pシリーズのパレット式マグネットチャックはERWA・3R・本間マルチチャックに対応し、放電加工機・研削盤・三次元測定機間でワークを再プリセットなしに移動できます。異なる加工機同士をつなぐことで、真のIoT連携（外段取り自動化）が実現します。
- YES → チャック系が原因ではない。機械系ブランチへ。

#### ブランチC ― 機械系

**Q1. 精度不良はシフトの経過とともに悪化するか（最初の部品はOK、最後の部品はNG）？**

- YES → 熱変位が主因。30分以上のウォームアップを実施し、主軸軸受温度とクーラント温度の安定性を確認する。
- NO → Q2へ。

**Q2. プリセッターテーブルのXY直角度は0.003mm以内か？**

- NO → プリセッター自体が外段取り作業に角度誤差を持ち込んでいる。SEP-02A(B)電極・ワークプリセッターはXY直角度0.003mm以内、各軸ヨーイング±0.001mm/150mmを仕様として保証しています。この値を超える場合は再校正または交換が必要です。
- YES → Q3へ。

**Q3. 丸物形状の場合、プリセッターの回転振れは0.002mm以内か？**

- NO → 中心調整の誤差が機械に転写されている。SEP-02A(B)に搭載できるSST-200回転プリセッターは、EROWA/system3Rマスターに対して回転振れ0.002mm以内を保証しており、外段取りで真円度測定と電極中心調整を正確に実施できます。
- YES → 機械本体の幾何精度に問題がある。全測定値を記録し、工作機械メーカーに検査を依頼する。

### 6. 診断文化を現場に根付かせる3つの習慣

フローチャートは、それを支える習慣があってこそ機能します。以下の3つを日常作業に組み込むことで、診断精度が大幅に向上します。

**第一に、ベースライン測定を標準化する。** 各シフト開始時にチャックのON/OFF保持力と残留磁気を記録します。所要時間は2分以内で、蓄積されたデータは品質不良が発生する前に劣化を察知するトレンドログになります。

**第二に、段取りと加工を分離する。** 外段取りとは、加工機を止めずに機外で準備作業を行うことです。SEP-02A(B)を使った外段取りでは、段取り誤差を機外で発見・修正できるため、機械の中でトラブルが発現することがありません。外段取りの導入で20%以上の生産性向上、段取り時間の大幅短縮が期待できます（カタログ実績値）。

**第三に、クランプ系統を加工機をまたいで統一する。** SM-Pシリーズのパレット式マグネットチャックを使えば、放電加工機でクランプしたワークをそのまま研削盤や三次元測定機に移動できます。EROWA・system3R・本間マルチチャックに対応しており、異なる加工機同士を「川の流れのように」つなぐことで、属人化した段取りから脱却できます。

## まとめ

この診断アプローチの基本は3つです。第一に、調整前に分類する。ワーク・チャック・機械のどの系統が原因かを特定してから行動することで、誤補正を防ぎます。第二に、数値基準を持つ。回転振れ0.002mm、直角度0.003mm、残留磁気5ガウスという具体的な合否基準が、主観的な判断を排除します。第三に、外段取りを品質の一次関門にする。SEP-02A(B)とSST-200回転プリセッターは、品質管理を機械サイクルの外側—修正が容易で観察可能なステップ—に移すために存在しています。
