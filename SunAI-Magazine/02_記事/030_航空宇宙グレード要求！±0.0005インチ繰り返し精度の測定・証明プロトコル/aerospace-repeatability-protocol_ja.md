# 航空宇宙グレード要求！±0.0005インチ繰り返し精度の測定・証明プロトコル

## 概要

AS9100 Rev DおよびISO 13485が求める±0.0005インチ（±0.0127mm）の繰り返し精度を、NIST/DKDトレーサビリティを確保しながら測定・文書化・証明する完全な手順を解説します。100回クランプ試験の実施方法、統計的工程管理（SPC）による評価、認証報告書の構成、そして株式会社Sun Ai製品による実測データを収録しています。

## 想定読者

- AS9100 / ISO 13485認証の取得責任を担う品質保証部門のマネージャー
- 航空宇宙・医療機器生産ラインのワーク保持設備仕様を策定する設計エンジニア
- ドイツ・アメリカの航空宇宙産業クラスターへの参入を目指す製造業者

## この記事で学べること

1. 航空宇宙認証において従来の繰り返し精度の主張が不十分な理由と、トレーサブルな証拠の生成方法
2. SPC分析を伴う100回クランプ試験の具体的な実施手順
3. 株式会社Sun Ai製品が航空宇宙グレードの精度をどのように達成し、文書化するか

## 本文

### 1. AS9100において「カタログ値」だけでは不十分な理由

航空宇宙・医療機器のサプライチェーンは、マーケティング上の主張ではなく客観的証拠を要求する品質フレームワークのもとで運営されています。AS9100 Rev D 条項8.5.1およびISO 13485 条項7.5.1はいずれも、ワーク保持を含む特殊工程について、国家または国際計量標準に準拠した文書化された統計的に意義あるデータによる妥当性確認を求めています。

「繰り返し精度0.002mm」というカタログ仕様は設計目標値であり、認証された性能主張ではありません。審査官は必ず「誰が、どの装置で、どの条件下で、どの国家標準にトレーサブルな形で測定したか」を問います。これらの質問に答えられなければ、その主張は適合の客観的証拠として使用できません。

株式会社Sun Aiの電極・ワークプリセッターSEP-02A(B)は、各軸のヨーイング±0.001mm/150mm（Y軸140mm）、XY軸間の直角度0.003mm以内、テーブル面とZ軸の直角度0.003mm以内を達成しています。回転プリセッターSST-200は、EROWAおよびsystem3Rマスターに対して回転フレ0.002mm以内を誇ります。これらの数値は達成可能です。しかし航空宇宙顧客に対しては、トレーサブルな基準器を使って測定・記録・承認されていなければなりません。

### 2. NIST/DKDトレーサブルな測定チェーンの構築

トレーサビリティとは、測定チェーン内のすべての計器が、米国のNISTまたはドイツのDKD（Deutscher Kalibrierdienst）ネットワークなどの国家計量機関にさかのぼる校正証明書を持つことを意味します。

±0.0005インチ（12.7µm）の繰り返し精度試験に推奨される計器構成は次のとおりです。分解能0.001mmのレーザー干渉計または校正済みダイヤルゲージ（ISO/IEC 17025認定試験機関による12か月以内の校正証明書付き）を一次センサーとして使用します。基準基面にはグレードA認定の花崗岩定盤（300mm上で平面度0.003mm）を使用します。環境は20℃±0.5℃の温度管理された空間で測定し、熱膨張誤差を排除します。

すべての計器の校正証明書を試験報告書に添付します。証明書番号、校正日、次回校正期限、認定機関のISO/IEC 17025認定番号を報告書ヘッダーに明記してください。

### 3. 100回クランプ試験の実施手順

以下のプロトコルはVDI/VDE 2617（三次元測定機の精度に関するガイドライン）を参考に、マグネットチャックのワーク保持向けに適応したものです。

**準備。** チャック面と試験片取付面をイソプロピルアルコールで清拭します。試験片を測定環境に置いてから30分間の熱ソーク時間を確保します。試験開始時と終了時の周囲温度・湿度・気圧を記録してください。

**基準測定。** 試験片（焼入れ鋼製ゲージブロックまたは精密研削済みキューブ）をマグネットチャックに搭載します。株式会社Sun AiのSWL-H190AB（ON状態、磁束密度0.06T）を作動させ、校正済みゲージで試験片の基準形体X・Y・Z座標を測定・記録します。この値が基準値となります。

**サイクリング。** チャックをOFFにして試験片を取り外し、同じ名目上の向きで再設置し、チャックをONに戻します。基準値からのX・Y・Z偏差を記録します。チャックやゲージのセットアップを調整せずに100サイクル繰り返します。

**安全上の注意。** SWL-H190シリーズのアンダーバー一式オプションを使用するとクランプスペースが2mmに縮小され、ワークの落下リスクが上がります。各搭載サイクルにおいてワークの接地の際には細心の注意が必要です。SWL-H190AB+は磁束密度ON 0.16T・OFF 0.008Tと保持力が高い反面、OFFの際に残留磁気が少し残ります。試験ログにこの点を記録してください。

**データ記録。** 100回分の測定値をスプレッドシートに入力します。各軸の平均値（X̄）・標準偏差（σ）・Cp・Cpkを算出します。±0.0005インチの航空宇宙認証においては、工程能力指数CpkがCpk ≥ 1.33（公差を±3σ境界とした4σ工程相当）以上であることが求められます。

### 4. 統計的工程管理（SPC）による評価

SPCは生データをプロセスの安定性と能力に関する陳述に変換します。必要なグラフは2種類です。X̄-R管理図（平均値・範囲）でクランプ工程が統計的管理状態にあることを確認し、工程能力調査で公差要件への適合を確認します。

管理限界線（UCL/LCL = X̄ ± 3σ/√n）の外にデータ点がなく、中心線の上下に連続7点が続くランや体系的なトレンドなどの非ランダムパターンが見られなければ、工程は管理状態にあるとみなされます。管理外シグナルが出た場合は、工程能力調査の前に原因を特定・除去する必要があります。

工程能力調査では、公差は±0.0005インチ（総幅25.4µm）です。100回試験から得たσを用いてCp = 25.4µm / (6σ)、Cpk = min[(USL − X̄), (X̄ − LSL)] / (3σ) を計算します。航空宇宙顧客は特殊特性でCpk ≥ 1.33、安全クリティカル形体でCpk ≥ 1.67を要求するのが一般的です。

株式会社Sun Aiの電極回転装置SER-01Aは、コレットインターフェースで回転ブレ0.002mm以内を達成します。ワイヤー放電加工のみと電極回転装置による追加工を比較した文書化済み試験では、穴の真円度が0.0044mmから0.0015mmに改善し、表面粗さがRa 1.1646µmからRa 0.1279µmに改善しました（Zeiss UPMC-850 CMM、Accretech Surfcom 1900SD3による測定）。これはCpk主張を審査の場で裏付けるための、ビフォー・アフターのデータパッケージの典型例です。

### 5. 認証報告書の構成

航空宇宙顧客に提出する最終認証パッケージには以下のセクションが必要です。

**表紙：** 設備型番・製造番号・試験日・試験技術者名・品質マネージャー承認サイン。

**測定システム分析（MSA/ゲージR&R）：** 測定システムの繰り返し性と再現性の合計が公差の10%未満（±12.7µmの公差に対しては1.27µm未満が理想）であることを証明します。

**校正証明書のコピー：** すべての測定計器のトレーサビリティチェーンを明記したもの。

**100サイクル生データ表：** 各サイクルのX・Y・Z偏差を記録。

**管理図：** 各軸のX̄-R図またはI-MR図で統計的管理状態を確認。

**工程能力指数：** 各軸のCp・Cpk、および顧客要求のCpk閾値に対する明確な合否判定。

**環境条件ログ：** 試験中の温度・湿度・異常事象。

株式会社Sun Aiの電極・ワークプリセッターSEP-02A(B)（本体重量220kg、サイズZ1500mm×Y650mm×X450mm、軸直角度0.003mm以内）は、この水準の文書化を繰り返し生成するための機械的安定性を提供します。オプションの回転プリセッターSST-200（回転フレ0.002mm以内）はSEP-02A(B)に直接搭載でき、1回のセットアップで直線方向と回転方向の両方を特性評価できます。

## まとめ

信頼性の高い航空宇宙グレードの繰り返し精度認証を定義する3つのポイントがあります。測定チェーンはISO/IEC 17025認定校正を通じてNISTまたはDKDにトレーサブルでなければならない。100回試験は管理された環境条件下で実施し、SPC（最低限Cpk ≥ 1.33）で評価しなければならない。そして生データ・管理図・工程能力指数・校正証明書を含む完全なパッケージを、AS9100審査官が追加説明なしにレビューできる構造化報告書としてまとめなければならない。株式会社Sun Ai製品が機械的精度の土台を提供し、本プロトコルがそれを証明する文書上の証拠を提供します。
