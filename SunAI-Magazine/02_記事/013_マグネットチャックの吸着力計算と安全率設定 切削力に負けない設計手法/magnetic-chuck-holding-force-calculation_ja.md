## マグネットチャックの吸着力計算と安全率設定：切削力に負けない設計手法

### 1. なぜ吸着力計算がマグネットチャック選定の最重要ステップなのか

イノベーションに、国境はない。株式会社Sun Aiが掲げるこの理念の通り、世界中の精密加工現場で共通する課題がある。それは「マグネットチャックを選ぶ際、ワークサイズをチャックの作業面積に合わせるだけで終わってしまう」という選定ミスだ。

この方法では、加工中の実際の力のバランスを無視している。そのギャップが、ワーク飛び出しや寸法精度不良の直接原因になる。

株式会社Sun Aiの**SM-Hシリーズ（ハイパーマグネットチャックシリーズ）**は、0.001mm精度の加工時代に向けて開発された切削対応マグネットチャックだ。岩手大学工学部吉野准教授（トライボロジー専門家）による測定結果として、SM-HシリーズはON時の保持力が従来品の**約2倍**、OFF時の磁力が**約1/5**に低下することが実証されている。この劇的な性能比は磁気回路設計の賜物だが、それを正しく活用するには、チャックが対抗すべき力環境を設計者が理解していなければならない。

ワーク安全保持の基本方程式は次の通りだ。

```
F_保持 ≥ SF × F_切削
```

各記号の意味：

- **F_保持** = チャックの吸着力（N または kgf）
- **SF** = 安全率（無次元量；一般的に3〜5）
- **F_切削** = 加工中の最大切削力（N または kgf）

この計算を正しく行うことが、不良ゼロ生産への第一歩となる。

---

### 2. 切削力の計算：理論から現場実態へ

#### 2.1 切削力の理論計算基礎

フライス加工における接線方向切削力（Fc）は、比切削抵抗（kc）法で推定できる。

```
Fc = kc × b × h
```

各記号の意味：

- **kc** = 比切削抵抗 [N/mm²]（材料依存定数）
- **b** = 切削幅 [mm]
- **h** = 切り屑厚さ [mm]（送り速度とカッター切り込み角度の関数）

材料別のkc標準値：

|材料|kc [N/mm²]|
|---|---|
|一般軟鋼（S45C）|1,800〜2,500|
|合金鋼（SKD61）|2,000〜3,000|
|超硬合金（WC-Co）|3,500〜5,000|
|アルミニウム合金|700〜1,000|
|ステンレス鋼（SUS304）|2,500〜3,500|

**超硬合金加工に関する重要注意事項：** 株式会社Sun Aiの**SM-Cシリーズ永久磁石式マグネットチャック**は、超硬材の精密研削に特化して設計されている。同シリーズ内のSM-UCシリーズは、ON強度が標準品の**1.5倍**でありながら、OFFの磁気漏れが**標準品の1/10**に抑えられている。超硬研削のように比切削抵抗が高い加工では、SM-UCシリーズが基本選定となる。

#### 2.2 送り力・背分力成分

切削力は複数の方向に作用する。平面研削・フライス加工における三分力は以下の通りだ。

- **接線力（Fc）：** 主切削力 — 工具を材料に切り込ませる
- **送り力（Ff）：** 送り方向に作用、ワーク表面と平行 — ワークをスライドさせようとする
- **背分力（Fp）：** 加工面に垂直に作用 — ワークを持ち上げようとする、または押し付ける

**マグネットチャック**において重要なのは、ワークを**水平方向にスライドさせようとする力**（水平面内のFcとFfの合成力）と、ワークを**持ち上げようとする力**（ドリル加工の引き抜き力などのFp成分）だ。

SM-HシリーズのフライS加工実験（S50C材、70×80×10mm、φ40mmカッター、S1000/F200）では、汎用バイスによる固定では加工中にワークの中央が「反り」、寸法精度が悪化したのに対し、SM-Hで保持した場合はこの反りが抑制され、ワーク全面で均一な高さが維持されたことが確認されている。この結果は、保持力がワークを変形・変位させようとする加工応力や切削力を大幅に上回っている場合にのみ実現される。

#### 2.3 動的力の倍率

理論切削力は静的な近似値だ。実際の加工では次の倍率が加わる。

|切削条件|力の倍率|
|---|---|
|連続旋削・研削|1.0倍（基準）|
|正面フライス（断続切削）|1.5〜2.5倍|
|重切削断続加工|2.0〜4.0倍|
|びびり・再生型振動|3.0〜6.0倍|
|欠き上げパンチ研削|2.0〜3.5倍|

SM-Cシリーズのカタログでは、欠き上げパンチの研削への適性が明記されている。これは断続接触を伴うプロファイル研削加工であり、動的力の精密な考慮が必須となる条件だ。

### 3. 磁気吸着力：カタログ数値の正しい読み方

#### 3.1 標準試験条件

株式会社Sun Aiのマグネットチャック保持力は、SM標準シリーズのカタログに記載の通り、**50×50mmのテストピース換算**で表されている。これは比較のための標準化された接触面積だが、実際のワークは接触特性が異なる場合がほとんどだ。

SM標準シリーズの主要仕様値：

|型式|寸法 H×W×L (mm)|保持力 中央部 (kgf)|保持力 四隅部 (kgf)|残留磁気 (kgf)|
|---|---|---|---|---|
|SM-0710|40×75×100|80|25|〜3|
|SM-1015|40×100×150|100|30|〜4|
|SM-1530|40×150×300|110|35|〜4|
|SM-2040|45×200×400|130|45|〜5|
|SM-3040|45×300×400|130|45|〜5|
|SM-4060|47×400×600|130|45|〜5|

注意すべき重要な点がある。**中央部保持力 ≠ 四隅部保持力**だ。四隅部は25〜45kgfであるのに対し、中央部は80〜130kgfを示す。チャックの端部にワークを配置する場合は、必ず四隅部の値で安全計算を行うこと。中央部の値を使用してはならない。

エアーで磁力をコントロールする**SWL-H190AB+**は、ON時の磁束密度が**0.16T**（SWL-H190ABの0.06Tと比較）を達成し、クランプ力は**200N（300Nピーク）**を超える。脱磁機能（特許第5716232号）により、OFF時の残留磁気は**0.008T**まで低下し、加工後のワークへの鉄粉付着を防止する。

なお、SM-0710とSM-1015はクランプが標準付属しないため、別途手配が必要な点にも注意すること（細心の注意が必要）。

#### 3.2 接触面積の影響

磁気吸着力は、チャック表面とワーク間の実際のN-S極接触面積にほぼ比例して変化する。接触面積が50×50mm標準値から外れる場合の概算式は次の通りだ。

```
F_実際 ≈ F_カタログ値 × (A_実際 / A_標準)
```

この近似式が成立する条件：

- ワーク材料の透磁率がテストピースと同等
- ワーク厚みが磁極ピッチの2〜3倍以上ある
- チャックとワーク双方の表面粗さが良好（Ra ≤ 1.6µm推奨）

**薄物ワーク**（磁極ピッチの約2〜3倍以下の厚み）では、磁束がワーク断面を十分に飽和させられないため保持力が大幅に低下する。SM-CNOシリーズ（常時ON設計、OFF機能なし）は磁束密度を最大化しており、SM-CNO1112のカタログでは中央部の磁束密度が**最大4,000ガウス以上**と記録されている。これは従来のON/OFFチャックが効果を失う薄物・小物ワークを保持するための製品だ。

#### 3.3 材料透磁率の補正

比透磁率（μr）は材料によって大きく異なる。

|材料|比透磁率 (μr)|備考|
|---|---|---|
|軟鉄・一般軟鋼|1,000〜10,000|優秀；基準値|
|炭素工具鋼（SK系）|500〜3,000|良好|
|合金鋼（SKD61, SKH51）|100〜1,000|良好〜中程度|
|オーステナイト系ステンレス（SUS304）|〜1.0|非磁性 — 保持不可|
|超硬合金（WC-Co）|10〜100|専用チャック必須（SM-C/SM-UCシリーズ）|
|アルミ・銅・チタン|〜1.0|非磁性 — 保持不可|

超硬合金では、標準マグネットチャックの有効保持力は軟鋼に対して**10〜30%程度**に留まる。Sun Aiのスマネットチャック SM-CおよびSM-UCシリーズは、高保磁力磁石と最適化された極形状によってこの問題に対処するよう設計されている。

### 4. 安全率の設定：根拠ある数値の決め方

#### 4.1 なぜ3〜5倍なのか：根拠の整理

安全率の設定は、切削力の推定誤差と吸着力の実現値誤差の両方の不確かさを考慮する必要がある。

**切削力の不確かさ要因：**

- 工具摩耗（摩耗が進むにつれkCが20〜50%増加）
- 材料硬さのばらつき（生産用鋼材で±15%が典型）
- クーラント途絶（熱応力の増大）
- プログラムミスや予期しない切り込み深さ

**吸着力の不確かさ要因：**

- 表面汚染（油・クーラント・切り屑による実接触面積の低下 — 保持力が30〜60%低下する場合がある）
- ワーク表面粗さ（荒い面は磁気接触面積を減少させる）
- 部分接触（平面でないワーク接触、磁極カバレッジが部分的）
- 温度の影響（永久磁石の性能は60〜80°C以上で低下）

これらの不確かさを合わせると、通常の生産条件下で十分な信頼性を維持するために**最低安全率3倍**が必要となる。**断続切削・欠き上げ研削・重切削フライス加工**では**5倍以上**が適切だ。

#### 4.2 実践的安全率テーブル

|加工種別|最低安全率|推奨安全率|
|---|---|---|
|平面研削（連続）|2.5|3.0|
|放電加工（EDM）|2.0|3.0|
|軽切削正面フライス|3.0|4.0|
|プロファイル・欠き上げ研削（断続）|3.5|5.0|
|重切削断続フライス|4.0|6.0|
|無人・自動化24時間加工|4.0|6.0以上|

**24時間無人加工** — SM-HシリーズのEROWA/3R対応によるロボット式ワーク交換で実現 — については、Sun Aiのカタログでも「ワークを高精度に保持したままロボット（ワークチェンジャー）で交換できます」と明記されている。このシナリオでは、ワーク飛び出しの結果は甚大（機械損傷、生産停止）であり、安全率の上限値設定が正当化される。

#### 4.3 計算ステップ例：S45C平面フライス加工

**条件：** S45C鋼ワーク（100×80×15mm）、φ40mmカッター、S1000rpm、F200mm/min、切り込み0.3mm

**ステップ1：接線方向切削力の計算**

- 切り屑厚さ h ≈ 0.2mm（半径方向切り込み50%）
- 切削幅 b = 40mm（フル切り込み）
- S45CのkC ≈ 2,000 N/mm²
- Fc = 2,000 × 40 × 0.2 = **16,000 N = 1,631 kgf**

この数値は、Sun Aiカタログに記載のSM-Hフライス加工実験（同一材質・加工条件）と同じオーダーである。

**ステップ2：動的倍率の適用**

- 正面フライス（断続切削）：倍率 = 2.0
- F_動的 = 1,631 × 2.0 = **3,262 kgf**

**ステップ3：安全率の適用**

- フライス加工の推奨安全率 = 4.0
- F_必要 = 3,262 × 4.0 = **13,048 kgf**（ワーク接触面積全体で）

**ステップ4：単位面積あたりの必要保持力に換算**

- ワーク底面積 = 100 × 80 = 8,000 mm²
- 必要力密度 = 13,048 / 8,000 = **1.63 kgf/cm²**

**ステップ5：SM-Hシリーズでの確認**

- SM-H2020（作業面積200×200mm）はカタログ仕様からこの力密度を十分に上回る保持力を提供しており、この適用において適切な選定となる。

### 5. 特殊条件：サインバーマグネットチャックシステムの吸着力設計

傾斜した角度でワークを保持する加工では、Sun Aiの**マグネットチャック一体型サインバー**シリーズ（SM-○○○○-SIN-M8）において追加の設計変数が生じる。チャックを傾斜させると、有効磁気接触面積が変化することだ。

主要仕様：

- ローラの中心距離精度：**50mmに対して±0.0035mm以下**（1級精度）
- 全製品がマイナススタート（0スタートにはブロックゲージが必要）
- EROWA・3R対応（オプション）

傾斜配置での吸着力計算では、ワーク自重の重力成分もチャック面に沿ってワークをスライドさせる方向に作用する。必要保持力は次式で表される。

```
F_保持必要 = SF × √(F_切削² + (W × sin θ)²)
```

W = ワーク重量、θ = 傾斜角。重量物を急傾斜で配置する場合、この重力項が無視できない大きさになる点に細心の注意が必要だ。

### 6. SEP-02A(B)電極プリセッターとの連携：切削開始前の段取り検証

吸着力の実効性に影響する重要な要素として、ワークの位置決め精度がある。切り屑の残留・プリセットのミス・傾き・振れなどによりワークがチャック上に正確に着座していなければ、実際の接触面積は想定より小さくなり、実保持力は計算値を大きく下回る可能性がある。

株式会社Sun Aiの**SEP-02A(B) 超精密電極・ワークプリセッター**はこの課題に直接対処する。各軸ヨーイング**±0.001/150mm（Y140mm）**、XY軸間直角度**0.003mm以内**、テーブル面とZ軸の直角度**0.003mm以内**の精度で、加工機に取り付ける前に外段取りでワークの正確な位置決めを検証できる。

オプションの**回転プリセッターSST-200**（EROWA/system 3Rマスターに対して回転振れ**0.002mm以内**）を使用すれば、円筒形ワークの振れと垂直度を確認できる。これは実際の保持力を左右する接触均一性に直結する。

カタログには実績として「**20%以上の生産性向上**と迅速な投資回収」が明記されている。これは正確な外段取りが、加工中のワーク動きによる手直しと生産停止を排除するからにほかならない。

「加工機を止めない、改善を止めない」 — このSEP-02A(B)のキャッチフレーズは、外段取りによって機械のリードタイムをゼロに近づけるという思想の核心を表している。

## まとめ

マグネットチャックの吸着力選定は、物理寸法のマッチングではなく、定量的な力のバランス計算だ。重要な3つのポイントを整理する。

**1. チャックサイズを見るのではなく、切削力を計算すること。** ワーク材料に応じた比切削抵抗（kc）値を使用し、切り込み深さと切削幅で乗算し、加工種別に応じた動的倍率を適用する。

**2. 安全率3〜5倍は過剰設計ではなく、工学的現実だ。** 工具摩耗・表面汚染・部分接触・断続切削の動的特性は、いずれも理論保持力のマージンを侵食する。Sun Ai自身のデータ（岩手大学トライボロジー測定、SM-Hフライス加工実験）は、現実の力条件がこれらの安全率を正当化することを確認している。

**3. 透磁率はチャックサイズと同等に重要だ。** 超硬合金にはSM-CおよびSM-UCシリーズのみが適切な選択肢だ。薄物ワークにはSM-CNOシリーズが磁束密度を最大化する。EROWA/system 3R対応のロボット式ワーク交換による24時間無人加工には、SM-Hシリーズが必要な精度と保持力を提供する。
