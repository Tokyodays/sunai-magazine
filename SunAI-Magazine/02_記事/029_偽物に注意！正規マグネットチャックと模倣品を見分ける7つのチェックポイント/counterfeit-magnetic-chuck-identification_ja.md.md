# 偽物に注意！正規マグネットチャックと模倣品を見分ける7つのチェックポイント

## 概要

正規品の30〜50%という価格で流通する模倣品マグネットチャックは、吸着力が最大40%低下し、製品寿命が正規品の3分の1程度にとどまる場合があります。加工精度と現場の安全に直結するこの問題を、調達・品質管理担当者が受入検査の段階で見抜くための7つのチェックポイントを解説します。

- 工具・治具の受入検査を担う品質管理部門の担当者

## 本文

### 1. 模倣品マグネットチャックが現場にもたらすリスク

外観が酷似していても、内部の永久磁石の材質や組み立て精度が劣る模倣品は、カタログに記載された保持力を発揮できません。研削盤や放電加工機でのワーク保持において、わずかなずれが加工不良や廃棄ロスに直結します。さらに、保持力不足によるワーク落下は重大な安全事故につながる恐れがあり、細心の注意が必要です。

株式会社Sun Aiの正規SMシリーズマグネットチャックは、モデルごとに保持力・残留磁気・手回しトルクをカタログに明記しています。たとえばSM-1530は、中央部110kgf・四隅35kgf・残留磁気約4kgfという具体的な仕様を公開しています。これらの数値を第三者測定で確認できない製品は、信頼性を疑うべき最初のシグナルです。

また、SWL-H190シリーズや SM-Hシリーズには特許第5716232号に基づく脱磁機能が組み込まれており、OFFサイクル後の残留磁気を最小化します。この機能を持たない模倣品では、加工後のワークに磁気が残り、後工程での鉄粉付着や測定誤差の原因となります。正規品の脱磁設計は現場の工程品質を守る重要な独自技術です。

### 2. 正規品と模倣品を見分ける7つのチェックポイント

**チェック1：型番と外形寸法・有効範囲の一致確認**

正規のSun Ai製マグネットチャックには、モデル名（例：SM-2040、SM-H1015、SM-C1580）と紐づいた寸法・有効範囲がカタログに掲載されています。SM-2040であれば外形45×200×400mm、有効範囲200×360mmです。実測値とカタログ値が数ミリ以上乖離している場合、または型番に対応するカタログが存在しない場合は模倣品の疑いがあります。

**チェック2：磁極ピッチと表面加工品質の確認**

正規品の磁極ピッチは用途別に設計されており、標準SMシリーズは鉄:真鍮=2:1（3mmピッチ）、SMハイパーシリーズは3:11（14mmピッチ）などモデルごとに規定されています。磁極面を十分な照明のもとで目視確認し、ピッチの均一性と表面の研削仕上がり品質を検査します。不均一なピッチや粗い表面仕上げは、精度加工機向けの製品として不適切です。

**チェック3：ガウスメーターによるON/OFF性能の実測**

最も客観性の高い確認方法です。正規品のSM-UCシリーズは、OFF時の磁力がON時の約1/10という設計値を持ち、スチール材の加工後に磁気が残りません。SWL-H190AB+では ON時0.16T・OFF時0.008T（テスラ単位）という実測データが公開されています。テスラメーターまたはガウスメーターを用いて表面磁束密度を計測し、公開値との一致を確認してください。一致しない場合、使用磁石材料の品質が劣る可能性が高いです。

**チェック4：ON/OFF切り替え機構の操作感確認**

正規SMシリーズには切り替え軸（レバー）が標準装備されており、規定の手回しトルク（例：SM-0710は20kg・cm）でスムーズに切り替わります。SM-1530より大きいサイズにはクランプが標準付属しますが、SM-0710とSM-1015はクランプ別売となっています。レバー操作にがたつき・引っかかり・過剰なトルクが必要な場合は、内部機構の品質が疑われます。

**チェック5：対象モデルへのプッシュアップピン付属確認**

SM-C1580とSM-C1009Vは、欠き上げパンチ研削に対応するプッシュアップピンが標準装備されています。模倣品ではこれらの機能部品が省略されているか、寸法精度が低く正しい着座が得られない場合があります。付属品リストとカタログを照合し、ピン本数・着座精度を確認してください。

**チェック6：EROWA / system 3R 対応精度の書類確認**

SMパレット式マグネットチャック（SM-Pシリーズ）やSTM電極ホルダーシリーズのEROwA / system 3R対応品は、各規格の位置決め精度に整合する加工精度が必要です。正規STMシリーズは、全製品の調整ネジに細目ボールプランジャを採用（2015年現在 世界で唯一）しており、従来品より20%調整しやすく、調整中のヨレや戻りが発生しません。調整ネジの操作にざらつきや戻りが感じられる場合、正規品ではない可能性があります。

**チェック7：適合証明書の要求とメーカーへの直接確認**

株式会社Sun Aiは岩手県奥州市に本社を置き、正規販売代理店の確認および製品の真正性照会に対応しています。正規品には製造元の保証と製造物責任保険が伴います。サプライヤーが株式会社Sun Aiの製造実績に紐づく書類を提示できない場合、または市場価格から30〜50%以上の大幅割引が提示される場合は、購入前に必ずSun Aiへ直接確認することを強くお勧めします。

### 3. 正規品が生み出す定量的な加工改善効果

正規品と模倣品の性能差は、加工データとして明確に現れます。SM-Hシリーズ（切削対応マグネットチャック）は、岩手大学工学部トライボロジー専門家の立会いのもと大学にて試験測定が行われました。材質S50C（70×80×10mm）のミリング加工試験では、SM-Hによる保持時の表面変位が+1〜+8ミクロン（単位：ミクロン）に抑えられたのに対し、汎用バイスを用いた場合は-10〜+12ミクロンに達しました。この平面度の差は、SM-Hがフライス加工中の「反ろう」という材料応力解放を抑制する設計によるものであり、低品質な磁石材料を用いた模倣品では再現できません。

また、電極回転装置SER-01Aを使用したワーク加工測定では、ワイヤー放電加工のみの穴の真円度が0.0044mm（表面粗さRa1.1646μm）であるのに対し、電極回転装置による追加工後は真円度0.0015mm（Ra0.1279μm）へ大幅に改善されることが実測データで確認されています。このレベルの精度は、回転ブレ0.002mm以内という厳密な製造品質なくしては達成できません。

調達コストを比較する際は、模倣品による加工不良1件・計画外の機械停止1回で生じる損失が、正規品との価格差を大きく上回ることを念頭においた判断が合理的です。

## まとめ

正規マグネットチャックの真正性を確認する上で最も重要な3点は、第一に保持力の実測値をカタログ仕様と照合すること、第二にEROwA / system 3R対応の位置決め精度を書面で確認すること、そして第三に岩手県の製造拠点に直接問い合わせ可能な正規販売ルートを通じて購入することです。模倣品は受入時には判別が困難な場合がありますが、不安定なクランプ・高い残留磁気・早期の摩耗という形でその差が必ず現れます。精密加工の現場では、こうしたリスクを許容する余裕はありません。

**仕様の不整合確認:**

- SM-C1530 という型番がカタログ内で混在している（一方では SM-C1580 のプッシュアップピン標準記載）。本記事では、カタログ写真・対応表と整合する SM-C1580 と SM-C1009V を正として記載。SM-C1530 の記述はカタログ上の誤記の可能性あり、公開前に株式会社Sun Aiに確認を推奨。
- SER-01A のカタログには「回転ブレは0.002以内」と「〜0.002」の2表記が混在。本記事では「0.002mm以内」に統一。
- closstable_2025.pdf（クロステーブル SCT-2301/SCT-3400）は本記事のテーマと無関係のため除外。
